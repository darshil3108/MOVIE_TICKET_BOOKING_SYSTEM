To Run This Project Use Following Step

1. Extract the zip file

2. Find requirement.txt (install the requirements)


3. Open terminal 
   Write command - python setup_project.py
                 - python setup_complete_project.py

4. if any error of module not fount install the required python modules/libraries

5. open terminal at manage.py
  - python manage.py makemigrations
  - python manage.py migrate
  - python manage.py createsuperuser
  - python manage.py runserver
